from scipy.stats import ttest_ind

group1 = water_0518[water_0518['性別'] == '男'][cols]

group2 = water_0518[water_0518['性別'] == '女'][cols]

ts, ps = ttest_ind(group1, group2)

for col, t, p in zip(cols, ts, ps):
        
    print(f'{col:>8}：\tt 值 = {t:.3f}，p 值 = {p:.3f}')

Knowledge_score：	t 值 = 2.623，p 值 = 0.009
Attitude_score：	t 值 = 0.175，p 值 = 0.861
Behavior_score：	t 值 = 0.417，p 值 = 0.677

[討論] 第一項因 p 值均小於 0.05，所以從統計的觀點來說，可看出男女生性別因素在「知識題總分」上有顯著的差異，而在「態度題總分」、「行為題總分」上則沒有。

# 載入統計模組 - 單因子變異數分析(One-way ANOVA)
from scipy.stats import f_oneway

group1 = water_0518[water_0518['居住地'] == '北部地區'][cols]

group2 = water_0518[water_0518['居住地'] == '中部地區'][cols]

group3 = water_0518[water_0518['居住地'] == '南部地區'][cols]

fs, ps = f_oneway(group1, group2, group3)

for col, f, p in zip(cols, fs, ps):
    
    print(f'{col:>8}：\tF 值 = {t:.3f}，P 值 = {p:.3f}')

Knowledge_score：	F 值 = 0.417，P 值 = 0.264
Attitude_score：	F 值 = 0.417，P 值 = 0.543
Behavior_score：	F 值 = 0.417，P 值 = 0.307

[討論] 三項因 p 值均大於 0.05，所以從統計的觀點來說，可看出居住地在「三大量表總得分的平均」上沒有顯著的差異。
